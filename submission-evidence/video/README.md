# Demo Video Evidence

Store video evidence details in this folder.

## Required

Create a file named `demo-video-link.md` with:

- Final unlisted YouTube URL
- Video duration
- Short feature checklist covered in the video

## Template

Use this structure:

```md
# Demo Video Link

- URL: https://youtube.com/...
- Duration: 00:00

## Features Shown

- Registration/Login
- Settings and language/theme changes
- Routines flow
- Offline usage and sync
- API/database proof
```
