# Release Evidence

Place signed build and release proof in this folder.

## Required

- Signed APK evidence screenshot/export

## Suggested Files

- `signed-apk-proof.png`
- `build-output-summary.md`

## build-output-summary.md Template

```md
# Release Build Summary

- Build type: Signed APK
- Version: x.y.z
- Build date: YYYY-MM-DD
- Notes: Any signing or release configuration notes
```

## Security Note

Do not commit keystore files or private signing credentials.
