# Submission Evidence

This folder contains all supporting evidence required for final submission.

## Required Items

- Demo video link documented in root `README.md`
- Signed APK evidence (screenshot or export proof)
- Feature screenshots:
  - Auth (login/register)
  - Home/dashboard
  - Settings
  - Routines
  - Offline and sync flow
- Google Play Console upload/publish screenshot (if applicable)
- API/database evidence screenshot from hosted service

## Folder Structure

- `screenshots/` - app and console screenshots
- `video/` - video link note and any export proof
- `release/` - signed APK evidence and release artifacts

## Recommended Naming

- `screenshots/auth-login.png`
- `screenshots/auth-register.png`
- `screenshots/home-dashboard.png`
- `screenshots/settings.png`
- `screenshots/routines.png`
- `screenshots/offline-sync.png`
- `screenshots/api-database-proof.png`
- `screenshots/play-console-upload.png`
- `release/signed-apk-proof.png`
- `video/demo-video-link.md`

## Final Check

Before hand-in, verify:

1. Root `README.md` has your final demo link (not placeholder text).
2. All evidence files are committed and visible in this folder.
3. Evidence filenames are clear and map to rubric items.
